🎵 Web Harmonium Pro

Web Harmonium Pro is an advanced, fully browser-based multi-instrument synthesizer. Built entirely with the native Web Audio API, it requires no external audio samples or plugins to run. It features a beautiful glassmorphism UI, continuous drones (Shruti box), and a multi-layering system allowing you to play multiple instruments simultaneously.

Developed by Pranav Garasiya.

✨ Key Features

🎹 8 High-Quality Synthesized Instruments:

Harmonium (Rich, detuned reeds)

Electric Piano

Strings

Bansuri (Flute with LFO vibrato)

Drawbar Organ

Synth Lead

Brass

Pluck Guitar

🎛️ Multi-Layering System: Select multiple instruments at once to create rich, layered soundscapes.

🌀 Continuous Drones (Shruti): Toggle independent background drone notes, essential for Indian classical music.

🎨 Dynamic Themes: 4 beautiful UI themes (Midnight, Sunset, Ocean, Forest) with neon visualizer LEDs.

🎚️ Studio Controls: Master Volume, Built-in Reverb (Convolver), Octave Shifting, and Sustain pedal support.

⌨️ Perfect DAW-Style Mapping: Play full 3 octaves directly from your computer keyboard without any overlapping keys.

🚀 Zero Dependencies (Audio): Uses 100% native Web Audio API oscillators and filters.

🛠️ Tech Stack

Frontend: HTML5, JavaScript (ES6+)

Styling: Tailwind CSS

Audio Engine: HTML5 Web Audio API

Icons: FontAwesome

🚀 How to Run (Local Setup)

Since this project relies completely on frontend technologies and the Web Audio API, setting it up is incredibly simple:

Clone the repository:

git clone [https://github.com/pranav-garasiya/WEB-HARMONIUM-PRO.git](https://github.com/pranav-garasiya/WEB-HARMONIUM-PRO.git)


Navigate to the folder:
Open the downloaded project folder on your system.

Run the app:
Simply open the index.html file in any modern web browser (Chrome, Edge, Firefox, Safari).

Note: Browsers require you to interact with the page before audio can be played. Click "Start Audio" to initialize the audio engine.
